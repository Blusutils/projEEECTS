# Blusutils EEE Projects

Projects with serious purposals, but not serious names. 

## What is EEE Projects?

Generally speaking, 
these are utility projects that in the right hands can be very powerful, 
but these projects look very unserious.

All of these supports extendation.

## List of EEE Projects 
|Project name/Alt name|Description|Link|
-|-|-
|alAAARM (An Alarm \~clock in~ Action)|Simple Windows Forms application what creates alarm clocks|https://github.com/Blusutils/alAAARM
|captuREEE (CaptureExp)|Motion capture implementation for VR|https://github.com/Blusutils/captuREEE
